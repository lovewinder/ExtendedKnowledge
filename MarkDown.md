# How to use md

## 标题

使用"#"
  
共六级标题
  
建议在"#"后加空格

## 列表

### 无序列表

在文字前加上"-"或"*"

* 1
* 2

### 有序列表

在文字前加"1.","2.","3."

1. x
2. y

## 引用

在文本前加入">"即可

> 这里是引用

## 图片与链接

### 图片

![pic name](pic url)

![Mou icon](http://mouapp.com/Mou_128.png)

### 链接

[链接名](链接 rul)"显示图片名"

[Baidu](http://baidu.com)

```

This is [an example][url] page for details.

第一个是链接文字,第二个是辨识链接的标签

[url]: http://www.baidu.com "Optional Title Here"

```

This is [an example][url] page for details.

[url]: http://www.baidu.com "Optional Title Here"

## 粗体与斜体

用两个"*"包含一段文本是粗体

用一个"*"包含一段文本是斜体

**粗体**

*斜体*
